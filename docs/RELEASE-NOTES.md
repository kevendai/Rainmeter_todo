# Release Notes

## 1.1.1 - 2026-06-27

- 日程管理底部去掉“已选择/编辑”的批量任务逻辑，改为当前日期统计、设置、刷新同步和新建日程。
- 左侧月历改为自绘卡片风格，选中日期、高亮今天和有日程日期的小圆点与整体玻璃风格统一。
- 日历筛选改为胶囊卡片样式，只保留全部日程、本地日历和 CalDAV 日历。
- 修复日历筛选胶囊按钮鼠标经过后变白、文字几乎不可见的问题。
- 右侧顶部删除“日程列表/周视图/月视图”，保留当前日期标题、时间范围和排序。
- 日程卡片调整为横向时间段、标题、地点和日历标签；编辑仍保留在单条日程右侧。
- 调整日程管理图标为蓝色日历图标配浅蓝背景，修正标题间距、底部按钮样式和 CalDAV 标签宽度，并移除左侧“显示选项”按钮。

## 1.1.0 - 2026-06-26

- 日程管理入口改为使用待办管理同款图标，磁贴不再单独显示新增按钮。
- 日程管理窗口改为接近设计稿的布局：顶部搜索、左侧月历和筛选、右侧日程卡片列表、底部操作栏。
- 支持本地日历和 CalDAV 日历的新建、编辑；未配置 CalDAV 凭据时只创建本地日程。
- 删除入口收进编辑窗口，周期 CalDAV 日程删除时区分整组删除和仅本机隐藏本次。

## 1.0.3 - 2026-06-26

- 管理界面移除冗余的“筛选”按钮，搜索框继续在输入时实时刷新，并向右调整以填补按钮移除后的空位。
- README 当前发布版本、打包命令和发布包链接更新到 `1.0.3`。
- 日程磁贴的设置入口图标改为齿轮。

## 1.0.2 - 2026-06-26

- 论文设置窗口改为两页布局：`论文同步` 页用于配置同步地址、账号、密码、测试登录和保存同步；`标题翻译` 页用于配置 Tencent Cloud SecretId、SecretKey、测试连接和保存凭据。
- 论文设置窗口尺寸收回到 `620x560`，交互方式与日程设置的分页按钮保持一致，避免同步凭据和翻译凭据挤在同一个长表单中。
- 论文同步凭据继续保存到 `paper-sync.secret`，标题翻译凭据继续保存到 `translation.secret`，不改变现有 DPAPI 加密和运行时读取逻辑。
- 设置窗口新增 `关于` 页，显示当前版本号和 Full/Lite 版本类型，并提供 `检查更新`。
- 检查更新会读取 GitHub tags 判断最新版本，并从仓库对应 `releases/v*` 目录中按当前版本类型下载 `rainmeter-desktop-widgets-full-*.zip` 或 `rainmeter-desktop-widgets-lite-*.zip` 到用户 Downloads 目录。
- 检查更新检测到新版本时会先询问是否更新；确认后才下载新版本包、自动解压并运行包内 `Install-Skins.ps1` 部署到当前 Rainmeter 安装目录。安装脚本部署后会自动重启 Rainmeter 并重新激活 Todo/Calendar 皮肤。
- Todo 磁贴设置按钮的提示文案从“翻译设置”改为“待办设置”，和当前设置窗口包含的内容保持一致。

## 1.0.1 - 2026-06-26

- 修复日程设置的“自动转入”规则列表在高 DPI 或字体度量变化时文字被截断的问题。规则列表现在按实际字体计算行高并垂直居中绘制，避免选中条和文字高度不一致。
- 改进待办编辑器的标签选择：新增/修改待办时可直接维护 `论文`、`日程`、`已读`、`自动归档` 等通用标签；标签区默认保持单行紧凑，展开时浮在备注区域上层，收起后继续编辑备注。
- 完善论文待办的阅读状态语义：手动完成论文标记为 `已读`，06:00 日切自动完成的旧论文标记为 `自动归档`，恢复论文时会移除这两个状态标签并刷新创建时间。
- 优化 06:00 日切和时间边界刷新：保留桌面交互流畅度，同时减少后台宿主进程频繁启动；电脑休眠、错过整点或重新加载皮肤后会在下一次检查补做日切。
- 完善今日日程到待办的一次性/系列转换：CalDAV 提醒映射为待办开始时间，日程结束映射为截止时间，转换备注会保留原始时间、地点和说明，会议链接优先作为可点击目标。
- 修复 CalDAV UTC 时间解析导致的 8 小时时差问题，并在更新遇到 stale ETag 时重新读取当前资源后再重试。
- 提升桌面磁贴渲染清晰度：部署后保持不透明窗口并启用 Rainmeter DPI 感知，减少高 DPI 桌面上的模糊显示。
